# Cricket Tournament Analyzer: Professional cricket analytics toolkit.

Author:thelkotolsantosh

Features
- Net Run Rate Calculator
- Tournament Table Analyzer
- Match Simulator
- CSV Analytics
- CLI Interface


Example Commands
Simulate match
python src/main.py --simulate
Load table
python src/main.py --table data/sample_points_table.csv
Calculate NRR
python src/main.py --nrr 200 20 180 20
